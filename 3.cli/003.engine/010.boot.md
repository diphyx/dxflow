### Usage

```bash [Terminal]
dxflow engine boot
```

### Subcommands

| Command | Description | Help |
|---------|-------------|------|
| [**up**](/docs/cli/engine/boot-up) | Start the engine | `dxflow boot up --help` |
| [**down**](/docs/cli/engine/boot-down) | Stop the engine | `dxflow boot down --help` |
