---
title: Docker login request
description: Request of login platform docker
---

### Endpoint

```bash
PUT /api/platform/docker/login/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "context": "",
    "password": "",
    "server": "",
    "username": ""
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    }
]
```

### Notes

- Platform Permission Required
