---
title: Key register request
description: Request of register engine key
---

### Endpoint

```bash
POST /api/engine/key/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "identity": "",
    "key": "",
    "permissions": []
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": {
            "identity": "",
            "permissions": [
                ""
            ]
        }
    }
]
```

### Notes

- Master Permission Required
