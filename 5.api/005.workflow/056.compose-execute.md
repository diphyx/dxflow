---
title: Compose execute request
description: Request of execute workflow compose
---

### Endpoint

```bash
PUT /api/workflow/compose/execute/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "args": [],
    "context": "",
    "detach": false,
    "env": [],
    "file": "",
    "identity": "",
    "index": 0,
    "path": "",
    "privileged": false,
    "service": "",
    "user": "",
    "workdir": ""
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": ""
    }
]
```

### Notes

- Workflow Permission Required
