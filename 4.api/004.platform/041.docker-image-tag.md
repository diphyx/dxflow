---
title: Docker image tag request
description: Request of tag platform docker image
---

### Endpoint

```bash
PUT /api/platform/docker/image/tag/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "context": "",
    "identity": "",
    "tag": ""
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    }
]
```

### Notes

- Platform Permission Required
