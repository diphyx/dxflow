---
title: Docker container inspect request
description: Request of inspect platform docker container
---

### Endpoint

```bash
GET /api/platform/docker/container/inspect/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Query Parameters

| Key | Default |
|-----|---------|
| context |  |
| identity |  |
| size | `false` |
