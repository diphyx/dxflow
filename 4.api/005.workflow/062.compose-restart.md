---
title: Compose restart request
description: Request of restart workflow compose
---

### Endpoint

```bash
PUT /api/workflow/compose/restart/
```

### Auth

```bash
Authorization: Bearer ${token}
```

### Body

```bash
Content-Type: application/json
```

```json [Json]
{
    "context": "",
    "file": "",
    "identity": "",
    "no_deps": false,
    "service": [],
    "timeout": 0
}
```

### Response

```json [Json]
[
    {
        "kind": "status",
        "payload": {
            "code": 0,
            "message": ""
        }
    },
    {
        "kind": "entity",
        "payload": ""
    }
]
```

### Notes

- Workflow Permission Required
