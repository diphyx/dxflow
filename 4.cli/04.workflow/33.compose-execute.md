---
title: Workflow Compose Execute 
description: Execute command in a compose workflow
---

### Usage

```bash
dxflow compose execute <IDENTITY> [flags] <SERVICE> <PATH> [--] [ARGS...]
```

### Options

```bash
      --context string   docker context
  -d, --detach           detach from the container
  -e, --env strings      environment variables of command
      --index int        index of the container in multiple replicas
  -p, --privileged       execute command in privileged mode
  -u, --user string      user to execute command as
  -w, --workdir string   working directory of command
```

### Global Options

```bash
  -C, --config-profile string   specify the config profile (default "default")
  -L, --log-level string        specify the log level [debug, error, disabled] (default "disabled")
  -N, --no-color                disable color output
```
