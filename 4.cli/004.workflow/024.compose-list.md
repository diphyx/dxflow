---
title: Workflow Compose List 
description: List compose workflows
---

### Usage

```bash [Terminal]
dxflow compose list [OPTIONS]
```

### Options

| Flag | Shorthand | Description | Type | Default |
|------|-----------|-------------|------|---------|
| `--context` |  | Docker context | `string` | `` |
| `--all` | `-a` | Show all workflows | `bool` | `false` |
| `--filter` | `-f` | Filter workflows | `strings` | `[]` |

### Global Options

| Flag | Shorthand | Description | Type | Default |
|------|-----------|-------------|------|---------|
| `--config-profile` | `-C` | Specify the config profile | `string` | `default` |
| `--log-level` | `-L` | Specify the log level | `string` | `disabled` |
| `--no-color` | `-N` | Disable color output | `bool` | `false` |
