### Usage

```bash
dxflow proxy prune
```

### Global Options

```bash
  -C, --config-profile string   specify the config profile (default "default")
  -L, --log-level string        specify the log level [debug, error, disabled] (default "disabled")
  -N, --no-color                disable color output
```
