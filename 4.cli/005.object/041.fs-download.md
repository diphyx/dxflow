---
title: Object Fs Download 
description: Download a filesystem object
---

### Usage

```bash [Terminal]
dxflow fs download <IDENTITY> [DESTINATION] [OPTIONS]
```

### Options

| Flag | Shorthand | Description | Type | Default |
|------|-----------|-------------|------|---------|
| `--force` | `-f` | Force creation if exists | `bool` | `false` |
