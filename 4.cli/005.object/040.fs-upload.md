---
title: Object Fs Upload 
description: Transfers local files to dxflow storage
---

### Usage

```bash [Terminal]
dxflow fs upload <SOURCE> [IDENTITY] [OPTIONS]
```

### Options

| Flag | Shorthand | Description | Type | Default |
|------|-----------|-------------|------|---------|
| `--force` | `-f` | Force creation if exists | `bool` | `false` |
